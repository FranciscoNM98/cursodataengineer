Type of made event.

One of the following values: 

| event           | definition                                        |
|-----------------|---------------------------------------------------|
| add to cart     | The user added a product to the cart              |
| checkout        | The user is in progress to pay for an order       |
| package_shipped | The order has been paid and is being shipped now  |
| page view       | An user simply visited the website                |
